## BlizkoMap

BlizkoMap es una plataforma que re diseñe para mi amiga la rata

##  Características

-  Mapa interactivo con geolocalización.
-  Búsqueda de lugares por nombre.
-  Clasificación por categorías.
-  Diseño responsivo para dispositivos móviles y escritorio.
-  Sistema de valoraciones y reseñas.
-  Información de horarios y detalles de cada lugar.
-  Galería de imágenes para cada ubicación.


## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript 
- OpenStreetMap
- Leaflet.js
- Supabase
- Responsive Design
